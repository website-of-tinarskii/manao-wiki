## Why Manao?

<CardGrid stagger>
  <Card title="Music Requests" icon="star">
    Let your viewers request songs directly from Twitch chat using commands, it even has a built-in music overlay!
  </Card>

  <Card title="Chat Overlays" icon="comment">
    A ready-to-use and flexible chat overlay, user can set their custom nickname too.
  </Card>

  <Card title="Soundboard" icon="rss">
    Trigger sound effects instantly using soundboard controller or via Twitch Channel Points.
  </Card>

  <Card title="Custom Commands" icon="setting">
    Create your own commands using to interact with chat, manage virtual currency, and more!
  </Card>

  <Card title="Open-Source" icon="open-book">
    Manao is completely open-source and free to use. You can even contribute to its development!
  </Card>
</CardGrid>
