## ทำไมต้องมะนาว?

<CardGrid stagger>
  <Card title="คำสั่งเพลง" icon="star">
    ให้ผู้ชมของคุณขอเพลงได้โดยตรงจากแชททวิชผ่านคำสั่งต่าง ๆ มาพร้อมกับโอเวอร์เลย์เพลงในตัว!
  </Card>

  <Card title="โอเวอร์เลย์" icon="comment">
    มีโอเวอร์เลย์แชทที่พร้อมใช้งานได้เลย คนดูสามารถตั้งชื่อเล่นที่กำหนดเองได้ด้วย
  </Card>

  <Card title="ซาวด์บอร์ด" icon="rss">
    เรียกใช้เสียงเอฟเฟกต์ได้ทันทีผ่านตัวควบคุมซาวด์บอร์ดหรือให้คนดูแลกผ่านแต้มช่อง (ตั้งค่าได้)
  </Card>

  <Card title="เพิ่มคำสั่งใหม่" icon="setting">
    ถ้าคำสั่งที่มีอยู่ยังไม่ถูกใจ ก็สร้างคำสั่งใหม่เองได้ด้วย!
  </Card>

  <Card title="โอเพ่นซอร์ส" icon="open-book">
    มะนาวบอต เป็นโปรแกรมโอเพ่นซอร์สและฟรีเขียนโดย @ทินนาร์สกี้ เอาไว้ให้สตรีมเมอร์รวม tools ไว้ที่เดียวง่าย ๆ ไม่ต้องไปล็อกอินมั่วซั่ว ถ้าชอบก็กดไลก์ ใช่กดแชร์ ดีกว่าก็คือกดโอน

  </Card>
</CardGrid>
