ฟีเจอร์ใหม่ล่าสุดใน Manao เวอร์ชัน 2.2.0 คือ **หน้าควบคุมข้อความตอบกลับอัตโนมัติ** ผู้ใช้สามารถตั้งค่าให้บอตตอบข้อความในแชทโดยอัตโนมัติเมื่อเจอคำหรือข้อความที่กำหนดไว้ ไกด์นี้จะมาสอนวิธีเพิ่มข้อความตอบกลับง่าย ๆ ถ้ายากก็ไม่ต้องทำ ฮรุกๆๆๆ

### การเพิ่มข้อความตอบกลับอัตโนมัติ

เพื่อสร้างข้อความตอบอัตโนมัติ ให้ทำตามขั้นตอนดังนี้:

1. ไปที่หน้า **Custom Replies Manager**

2. กดปุ่ม `Add Reply` ที่มุมบนขวาของตาราง

3. กรอกรายละเอียดดังนี้:

   * **Keyword Type**: เลือกวิธีตรวจสอบข้อความ

      * `includes`: ตอบเมื่อข้อความมีคำหรือวลีที่กำหนด (เช่น includes "ไก่" -> "ไก่แจ้", "ไก่งวง" จะเข้าข่ายหมด)
      * `equals`: ตอบเมื่อข้อความตรงกับคำหรือวลีที่กำหนดเป๊ะ ๆ (เช่น equals "ไก่" -> "ไก่แจ้", "ไก่งวง" ไม่ได้ ต้อง "ไก่" อย่างเดียว)
   * **Keywords**: ใส่คำหรือวลีที่ต้องการให้บอตตอบ
   * **Response Type**: เลือกวิธีการตอบ

      * `random`: เลือกข้อความตอบแบบสุ่ม
      * `sequence`: ตอบข้อความตามลำดับวนกลับเมื่อถึงท้ายรายการ
   * **Responses**: ข้อความตอบที่บอตจะส่งเมื่อเจอคำหรือวลี (คั่นด้วยเครื่องหมายคอมม่า)

4. กดปุ่ม `Save` เพื่อบันทึกคำตอบใหม่

> สามารถแก้ไขหรือลบคำตอบเก่าได้จากปุ่มในตาราง

---

### การทดสอบ Custom Reply

หลังจากบันทึกแล้ว ทดสอบได้โดยส่งข้อความในแชท:

1. พิมพ์ข้อความที่ตรงกับ Keywords ที่ตั้งไว้
2. บอตจะตอบข้อความที่ตั้งค่าไว้โดยอัตโนมัติ
3. สำหรับ Response Type แบบ `sequence` ลองส่งข้อความซ้ำหลายครั้ง จะเห็นข้อความตอบวนตามลำดับ

### เคล็ดลับ & วิธีแก้ปัญหา

* **อัปเดตทันที**: การแก้ไขใน Custom Replies Manager จะอัปเดตบอตทันที ไม่ต้องรีสตาร์ท Manao
* **ไม่สนใจตัวพิมพ์ใหญ่/เล็ก**: การตรวจสอบข้อความไม่แยกตัวพิมพ์ใหญ่-เล็ก
* **หลาย Keywords**: สามารถตั้งให้ตอบข้อความเดียวกันได้หลายคำ
* **ข้อความว่าง**: ตรวจสอบให้ Response ไม่ว่างเปล่า มิฉะนั้นบอตจะไม่ตอบ

ถ้าบอตไม่ตอบข้อความ:

1. ตรวจสอบว่าบอตเชื่อมต่อกับช่อง Twitch ถูกต้อง
2. ตรวจสอบข้อความและ Keywords ว่าตรงกัน
3. ตรวจสอบว่าบัญชีบอตมีสิทธิ์ส่งข้อความในแชท
4. ถ้ายังมีปัญหาอยู่ สามารถเข้าไปสอบถามใน [Discord](https://discord.gg/vkW7YMyYaf)
